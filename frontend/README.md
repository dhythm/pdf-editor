# Frontend for PDF Editor

## Create environment

```sh
npm create vite@latest frontend -- --template react-ts
```

```sh
cd frontend
npm install @pdfme/generator @pdfme/ui @pdfme/common
```

## Run App

```sh
npm install
npm run dev
```

```sh
npm install react-router-dom
npm install eslint-plugin-unused-imports --save-dev
```
